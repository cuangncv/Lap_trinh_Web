# Photo Sharing App

Photo Sharing App là một ứng dụng chia sẻ ảnh gồm frontend và backend.

Frontend được xây dựng bằng React.  
Backend được xây dựng bằng Node.js và Express.  
Database sử dụng MongoDB.

---

## How to Run Project

### Clone repository

---

### Backend
 
npm install  
npm start  

Tạo file .env trong thư mục be

DB_URL=your_mongodb_connection_string  

Backend chạy tại http://localhost:8080

---

### Frontend

npm install  
npm start  

Frontend chạy tại http://localhost:3000

---

## Features

User authentication  
Upload photos  
View photos  
Comment and like photos
